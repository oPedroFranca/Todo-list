# Todo-list

```bash
# Clonar repositório
  $ git clone https://github.com/oPedroFranca/Todo-list.git
  
# Entrar na pasta do projeto
  $ cd Todo-list
  
# Instalar dependências
  $ npm install

# executar o projeto
  $ npm run dev
  
# entre na URL retornada (http://localhost:XXXX/)
```
